OFICINA DE PROGRAMAÇÃO (CRONOGRAMA DE AULAS)


CICLO 1 (GITHUB):
* Criação de conta no Github.
* Clonagem de repositório, pull e push requests
* Início do projeto: Ponto de Vendas Diretas
* Descrição: Neste ciclo os alunos aprenderão os conceitos iniciais de repositórios e colaboração de projetos. Cirarão um branch do projeto Ponto de Vendas e iniciarão sua codificação usando C# e Winforms.
Treinarão os conceitos de Programação Assíncrona e Paralela e Programação Orientada a Objetos indispensáveis ao desenvolvedor profissional e padrão no mercado de trabalho.

CICLO 2 (REDES):
* Protocolos HTTP e WebSocket
* Comunicação em Tempo real
* Servidores
* APIs
* Descrição: Neste módulo os alunos aprenderão a fazer a aplicação se comunicar através da rede, aprenderão sobre os protocolos de rede utilizados no mercado e criarão a API que será utilizada pela aplicação para cadastro e alteração de dados no servidor enquanto aprendem a função dos Endpoints de servidor. Aprenderão ainda os conceitos de segurança cibernética de servidores e proteção contra ataques de negação de serviço.

CICLO 3 (AUTH):
* Autenticação.
* Autorização.
* Descrição: Neste módulo os alunos aprenderão os conceitos de autenticação (identidade) e autorização (permissão) e aprenderão a criar um sistema de cadastro de usuários e controle de permissões para os funcionários que utilizarão a aplicação.

CICLO 4 (SQL):
* Descrição: Neste módulo os alunos aprenderão a criar a base de dados da aplicação com Microsoft SQL Server, converter dados JSON para tabelas SQL e armazenar os dados dos clientes, produtos e vendas no servidor de forma permanente.
* Ainda aprenderão os conceitos de segurança cibernética de servidores e proteção contra ataques de SQL Injection.


CICLO 5 (Criptografia):
* Hash.
* Criptografia.
* Descrição: Neste módulo os alunos aprenderão a criptografar dados sensíveis no servidor, armazenar e trocar chaves de criptografia seguras e aprenderão a criptogravar senhas de forma irreversível e como efetuar a verificação de usuários sem ver a senha do usuário.

CICLO 6 (Revisão):
* Descrição: Neste módulo os alunos revisarão a aplicação, publicarão ela de forma temporária, aprenderão a acessá-la e utilizá-la e ainda aprenderão as melhores práticas de atendimento ao cliente.
* Neste módulo os alunos aprenderão a função do desenvolvedor no mundo real e como ele constrói o futuro por meio de algoritmos.
